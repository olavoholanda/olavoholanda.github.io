Gamified Education Platform

MeuTutor is an adaptive and gamified education platform. The system helps brazilian students to prepare for the High School National Exam, using intelligent tutoring systems and Werbach's PBL Triad gamification theory. Its backend was developed in Java with SOAP Web Services and MySQL database accessed with Hibernate. Its frontend was developed using PHP, Javascript, HTML and CSS3. I've lead the backend team while sometimes maintained the fronted code.

Scholarship Contest Platform

This platform acts like a selective process for scolarships, providing custom tests for each school and a student performance report. Therefore, as higher the students performance, better will be his discount in the school enrollment. The system was developed with its backend using Java, Rest Services, Spring Boot, Hibernate and MySQL. At the frontend side we used Javascript, Angular, HTML and Less/Sass.

Intelligent Education Mobile App

I was part of a team that developed the mobile app version of an education platform. It prepares students for universities entrance exams with video lectures and questions for training. The app used the platform backend with rest services developed in Java. We used Ionic with Cordova to build the app for Android and iOS.

Java Ontology Integrated Toolkit

JOINT is an Object-Triple Mapping system that map RDF triples into data objects, enabling developers to handle RDF triples as objects in object-oriented applications based on ontologies. This project uses extensively Java Reflection API, SPARQL (Simple Protocol and RDF Query Language) and Sesame API. Its goal is to act like Hibernate for rdf triple store tools.
